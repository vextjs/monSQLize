---

## 🆕 新功能

### ✨ Model 层

- Schema 验证（集成 schema-dsl）
- 自定义方法（instance + static）
- 生命周期钩子（before/after）
- 索引自动创建
- 枚举配置支持
- 关系定义（预留接口）

**核心 API**:
- `Model.define()` — 注册 Model 定义
- `msq.model()` — 获取 Model 实例
- `validate()` — 数据验证
- instance 方法 — 注入到查询结果
- static 方法 — 挂载到 Model 实例

### ✨ Model 自动时间戳

- 自动管理 `createdAt` / `updatedAt`
- 支持自定义字段名
- 支持部分启用（只 createdAt 或只 updatedAt）

**已知限制**:
- `incrementOne` 暂不支持自动时间戳
- 用户手动设置的时间戳会被覆盖
