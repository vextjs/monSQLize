- ✅ 性能考量说明
- ✅ 5 个常见问题 FAQ

**代码示例**:
```javascript
// ✅ v1.3.0+: 自动转换 ObjectId 字符串
const posts = await collection('posts').find({
    authorId: '507f1f77bcf86cd799439011',      // 自动转换

});

// ❌ v1.3.0 之前: 需要手动转换

});
```
