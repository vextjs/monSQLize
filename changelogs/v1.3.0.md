# v1.3.0 — 2026-04-27

## 新功能：链式池访问 API

### 概述

新增 4 个公开方法，支持多连接池 / 多数据库路由，配合 vext 插件使用可实现 `app.db.pool('cn').use('billing').model('Invoice')` 的完整链式调用。

### 新增方法

#### `scopedCollection(collectionName, opts?)` — 限定路由 collection

```js
msq.scopedCollection('invoices')                         // 默认池 + 默认库
msq.scopedCollection('invoices', { database: 'billing' }) // 默认池 + billing 库
msq.scopedCollection('invoices', { pool: 'cn' })          // cn 池 + 默认库
msq.scopedCollection('invoices', { pool: 'cn', database: 'billing' }) // cn 池 + billing 库
```

#### `scopedModel(key, opts?)` — 限定路由 Model（不走 _modelInstances 缓存）

```js
msq.scopedModel('Invoice', { pool: 'cn' })
// connection 合并语义：opts 优先，definition.connection 作 fallback
// 例：Invoice 已绑定 connection.database='billing'，
//    调用 scopedModel('Invoice', { pool:'cn' }) → 自动路由 cn 池 billing 库
```

#### `pool(poolName)` — 连接池访问器

立即校验（`NOT_CONNECTED` / `NO_POOL_MANAGER` / `POOL_NOT_FOUND`），返回 PoolAccessor：

```js
const cn = msq.pool('cn')
cn.collection('orders')          // cn 池默认库
cn.model('Order')                // cn 池默认库
cn.use('billing').collection('invoices')  // cn 池 billing 库
cn.use('billing').model('Invoice')        // cn 池 billing 库
```

#### `use(dbName)` — 数据库切换访问器

适用于单连接多库场景，返回 ScopedAccessor：

```js
const billing = msq.use('billing')
billing.collection('invoices')   // 默认池 billing 库
billing.model('Invoice')         // 默认池 billing 库
```

### TypeScript 类型

`types/monsqlize.ts` 新增：
- `scopedCollection()` / `scopedModel()` 方法签名
- `pool()` 返回值类型（含 `use()` 嵌套）
- `use()` 返回值类型

### 错误码

| 错误码 | 触发条件 |
|--------|---------|
| `NOT_CONNECTED` | `dbInstance` 为 null（未调用 connect()）|
| `NO_POOL_MANAGER` | `_poolManager` 未配置 |
| `POOL_NOT_FOUND` | 指定 pool 名不存在；`err.available` 含可用池列表 |
| `MODEL_NOT_DEFINED` | `scopedModel` 中 key 未注册 |

### 测试

新增测试分组 6（`use()` 顶层访问器，4 个用例）和分组 7（`pool()` 顶层访问器，7 个用例），追加至 `test/unit/model/scoped-access.test.js`。

### 向后兼容

- `model()` / `collection()` / `pool()` / `use()` 均为**纯新增方法**，不影响已有 API
- `scopedCollection` / `scopedModel` 为内部公开化，原私有路由逻辑不变
