# 01 - 项目信息

> 版本: v1.2.1 | 更新: 2026-04-24

## 基本信息

| 项目 | 说明 |
|------|------|
| 项目名称 | monSQLize |
| npm 包名 | `monsqlize` |
| 当前版本 | v1.2.1 |
| 项目根目录 | `e:\Worker\monSQLize\` |
| 发布目标 | npm 公开包 |
| 模块格式 | CommonJS (`lib/index.js`) + ESM (`index.mjs`) 双模块 |
| 类型定义 | TypeScript (`index.d.ts`) |
| Node.js | `>= 16.0.0` |
| 许可证 | MIT |

## 核心依赖

| 包 | 版本 | 必须 | 说明 |
|----|------|:----:|------|
| mongodb | ^6.17.0 | ✅ | 核心驱动，兼容 4.x/5.x/6.x/7.x |
| schema-dsl | ^1.1.5 | ✅ | Model 层 Schema 定义 |
| ssh2 | ^1.17.0 | ✅ | SSH 隧道连接 |
| async-lock | ^1.4.1 | ✅ | 本地并发锁 |
| ioredis | ^5.8.2 | ❌ 可选 | L2 缓存层（optionalDependencies） |

## 本地命令

| 命令 | 说明 |
|------|------|
| `npm test` | 运行全部单元测试（`node test/run-tests.js`） |
| `npm run test:coverage` | 测试 + 覆盖率（nyc） |
| `npm run lint` | ESLint 检查（v9 Flat Config） |
| `npm run lint:fix` | ESLint 自动修复 |
| `npm run type-check` | TypeScript 类型检查（noEmit） |
| `npm run benchmark` | 性能基准测试 |

## 目录结构

```
monSQLize/
├── lib/                        # 核心源码（CommonJS）
│   ├── index.js                # 主类 MonSQLize（构造器、connect、model、pool API）
│   ├── connect.js              # ConnectionManager（仅 mongodb 已实现）
│   ├── model/                  # Model 层
│   │   ├── index.js            # Model 基类 + ModelInstance
│   │   └── features/           # softDelete/populate/relations/version/virtuals/defaults
│   ├── infrastructure/         # 连接池基础设施
│   │   ├── ConnectionPoolManager.js
│   │   ├── PoolSelector.js     # 5 种选择策略
│   │   ├── PoolConfig.js       # 配置验证
│   │   ├── HealthChecker.js
│   │   ├── PoolStats.js
│   │   ├── ssh-tunnel.js / ssh-tunnel-ssh2.js
│   │   └── uri-parser.js
│   ├── mongodb/                # MongoDB 适配层（唯一合法的原生操作入口）
│   ├── cache.js                # L1 内存缓存（LRU）
│   ├── redis-cache-adapter.js  # L2 Redis 缓存
│   ├── multi-level-cache.js    # L1+L2 组合
│   ├── transaction/            # 事务管理
│   ├── lock/                   # 分布式锁
│   ├── saga/                   # Saga 协调器
│   ├── sync/                   # ChangeStream 同步
│   ├── expression/             # 统一表达式系统（122 操作符）
│   ├── errors.js               # 统一错误系统
│   ├── logger.js               # Logger 工厂
│   └── operators.js            # 操作符映射（含 MySQL 占位）
├── types/                      # TypeScript 类型定义（不含 .js 实现）
│   ├── monsqlize.ts            # 主类接口
│   ├── pool.ts                 # 连接池类型
│   ├── model/                  # Model 类型（definition/instance/relations/virtuals）
│   └── ...
├── index.d.ts                  # 类型汇总入口
├── index.mjs                   # ESM 入口（代理 lib/index.js）
├── test/                       # 测试（run-tests.js 自定义运行器）
├── examples/                   # 使用示例
└── .devcodex/                  # AI 工作目录（.memory 已排除 git）
```
